Namespace PFVR.Player.Gestures

Classes

AbstractDetector

The base class for all complex (i.e., two-hand) gestures.

Implementing scripts call TurnOn() when they detect that hands have entered their trigger state, and TurnOff() when they leave it.

BasicDetector

An example implementation. Detects when hands are within a certain distance towards each other.

ClapDetector

A script to detect when hands are in a clapping position, i.e., palms facing each other and touching.

HandGatheringDetector

A script to detect when hands are in a Kame-Hame-Ha-like position, i.e., palms facing each other and 10-20 cm apart.
